## 4. 支持手工上传的表格


## 4.1 上传处理规则

1. 上传必须基于标准模板。
2. 上传前先校验模板表头、必填字段、字典值和主键。
3. 上传采用 `有错整批退回`，不建议允许“部分成功”。
4. 每次上传必须生成批次号并保留回执。
5. 每次失败必须提供失败明细下载。
6. `预校验失败`、`入库校验失败`、`无权限上传` 统一在上传工作台闭环，不进入异常中心。


## 4.1.1 手工上传预校验汇总表

| 展示编码 | 错误大类 | 场景说明 | 提示信息 | 对应错误码 |
| --- | --- | --- | --- | --- |
| `FILE` | `文件级错误` | 文件格式不正确、文件加密、空文件、超大文件、文件损坏 | 上传失败：当前文件无法被系统正常识别，请使用标准模板文件重新上传。 | `UPL-FILE-001`、`UPL-FILE-002`、`UPL-FILE-003`、`UPL-FILE-004`、`UPL-FILE-005` |
| `TPL` | `模板级错误` | 缺少必需 Sheet、表头不匹配、必填列缺失、重复列、不支持的单元格结构 | 上传失败：当前文件不是系统要求的标准模板，请下载最新模板并按模板结构填写。 | `UPL-TPL-001`、`UPL-TPL-002`、`UPL-TPL-003`、`UPL-TPL-004`、`UPL-TPL-005` |
| `ROW` | `行级错误` | 必填字段缺失、字段格式错误、枚举值不匹配、字典值不存在、文件内重复数据 | 上传失败：当前文件存在字段填写问题，请根据失败明细逐条修正后重新上传。 | `UPL-ROW-001`、`UPL-ROW-002`、`UPL-ROW-003`、`UPL-ROW-004`、`UPL-ROW-005`、`UPL-ROW-006`、`UPL-ROW-007`、`UPL-ROW-008`、`UPL-ROW-009` |

说明：

1. 预校验失败后，当前批次 `整批退回`，不进入正式业务库。
2. 当前业务口径为 `不允许部分成功、部分失败`。

## 4.2 上传错误码

### 4.2.1 文件级错误

| 错误码 | 场景 | 详细规则说明 |
| --- | --- | --- |
| `UPL-FILE-001` | 文件格式错误 | 上传文件必须为系统指定的 Excel 模板文件，建议统一为 `.xlsx` |
| `UPL-FILE-002` | 文件加密 | 不允许设置打开密码、编辑密码、权限保护 |
| `UPL-FILE-003` | 文件为空 | 文件中除表头外，至少存在 1 行有效业务数据 |
| `UPL-FILE-004` | 文件超大小限制 | 单文件大小不得超过系统限制，当前建议值为 `50MB` |
| `UPL-FILE-005` | 文件损坏 | 文件必须可正常打开和解析，不允许损坏文件 |

### 4.2.2 模板级错误

| 错误码 | 场景 | 详细规则说明 |
| --- | --- | --- |
| `UPL-TPL-001` | 缺少必需 Sheet | 模板要求的 Sheet 必须完整存在，名称必须一致 |
| `UPL-TPL-002` | 表头不匹配 | 表头名称、列顺序、列数必须与模板定义一致 |
| `UPL-TPL-003` | 必填列缺失 | 模板定义为必填的列不得删除、隐藏或改名 |
| `UPL-TPL-004` | 重复列 | 同一业务字段不得出现多列 |
| `UPL-TPL-005` | 不支持的单元格结构 | 不允许重复列、异常合并单元格、隐藏关键列、异常公式结构 |

### 4.2.3 行级字段错误

| 错误码 | 场景 | 详细规则说明 |
| --- | --- | --- |
| `UPL-ROW-001` | 必填值为空 | 各模板定义为必填的字段在去除前后空格后不得为空；必填字段范围以模板定义为准 |
| `UPL-ROW-002` | 字段长度非法 | 各字段必须满足预设长度限制 |
| `UPL-ROW-003` | 手机号格式错误 | 默认要求为 `11 位数字`，不得包含非法字符 |
| `UPL-ROW-004` | VIN 格式错误 | 默认要求为 `17 位`，不得包含非法字符，且不允许 `I/O/Q` |
| `UPL-ROW-005` | 日期格式错误 | 日期必须能解析为合法日期 |
| `UPL-ROW-006` | 金额格式错误 | 金额必须为合法数值；关键金额字段不得为负数 |
| `UPL-ROW-007` | 枚举值非法 | 客户类型、性别、联系偏好、关系标识、电话标签、邮箱标签、地址标签、标签分类、分群分类、车辆状态、签单状态、项目来源、权益类型、权益状态、服务类型、服务状态、保单类型、沟通方式、沟通结果、参与状态、操作动作必须命中枚举确认文件定义的标准枚举值 |
| `UPL-ROW-008` | 文件内重复行 | 同一批文件内不得出现重复主键、重复业务单号、重复 `VIN-合同号` 组合记录 |
| `UPL-ROW-009` | 字典值不存在 | 门店、顾问、车型标准名、保险公司、金融机构、机构代码必须能在主数据字典中找到 |

## 5. 异常中心模块、场景与编码

## 5.1 客户确认视角的异常分类总览

本节按客户页面展示视角整理异常中心内容。异常中心仅用于 `异常识别`、`异常展示`、`筛选查询`、`导出明细`、`责任分发`、`状态跟踪`，不承担 `直接数据修复`、`自动合并`、`自动拆分`、`规则裁决`、`来源系统回写` 的职责。

### 5.1.1 异常中心汇总表

| 展示编码 | 异常大类 | 场景说明 | 建议客户提示信息 | 对应系统编码 |
| --- | --- | --- | --- | --- |
| `VAL` | `有效性异常` | 字段格式不正确、字段值超出允许范围、字典值无法匹配 | 系统发现当前字段内容不符合要求，请先核对原始单据或来源系统数据，再由对应系统修正。 | `EX-VAL-001`、`EX-VAL-002`、`EX-VAL-003`、`EX-VAL-004`、`EX-VAL-005`、`EX-VAL-006`、`EX-VAL-007`、`EX-VAL-008`|
| `UNI` | `唯一性异常` | 疑似重复客户 | 系统发现当前记录存在重复，请人工确认后通知来源系统处理；系统不会自动合并。 | `EX-UNI-010` |
| `CMP` | `完整性异常` | 有客户姓名但是没有联系方式 | 系统发现当前记录信息不完整，页面暂时无法完整展示，请补齐来源数据后等待同步更新。 | `EX-CMP-001`、`EX-CMP-002`、`EX-CMP-003`、`EX-CMP-004`、`EX-CMP-005` |
| `REL` | `关联性异常` | 业务记录无法挂接至客户主体（如订单无法关联到客户） | 系统发现当前业务记录无法建立关联关系，请先核对 `VIN` 等关键信息，确认主体档案是否完整。 | `EX-CMP-009`、`EX-CMP-010` |


## 5.2 异常编码明细

### 5.2.1 有效性异常

| 错误码 | 场景 | 异常出现方式 |
| --- | --- | --- |
| `EX-VAL-001` | 联系电话格式错误 | 已入库购车人联系电话或送修人联系电话不是合法电话格式，或包含字母、空格、短横线 |
| `EX-VAL-002` | VIN 格式错误 | 已入库车辆 VIN 不是 `17 位`，或包含 `I`、`O`、`Q` |
| `EX-VAL-003` | 邮箱格式错误 | 已入库邮箱字段缺少 `@`、域名后缀，或整体不符合邮箱结构 |
| `EX-VAL-004` | 日期值非法 | 已入库日期字段无法解析为合法日期 |
| `EX-VAL-005` | 金额值域错误 | 已入库金额字段为负数、超出上限，或出现不允许为 `0` 的金额字段值为 `0` |
| `EX-VAL-006` | 枚举值非法 | 已入库字段值不在枚举确认文件约定的标准枚举值内 |
| `EX-VAL-007` | 字段长度非法 | 已入库姓名、公司字段、合同号、保单号、活动编码长度超出字段长度限制 |
| `EX-VAL-008` | 字典值不存在 | 已入库门店、顾问、车型标准名、保险公司、金融机构无法命中主数据字典 |


### 5.3.3 唯一性异常

| 错误码 | 场景 | 异常出现方式 |
| --- | --- | --- |
| `EX-UNI-010` | 相似客户信息冲突 | 根据需求规格说明书的运算规则匹配，并在异常中心形成一条异常记录，供业务人员人工确认；系统不自动执行合并、拆分、去重处理 |


### 5.3.4 完整性异常

| 错误码 | 场景 | 异常出现方式 |
| --- | --- | --- |
| `EX-CMP-001` | 客户缺联系方式 | 客户主体已存在，但 `联系电话`、`邮箱`、`联系地址` 全部为空 |
| `EX-CMP-002` | 业务记录缺归属门店 | 销售记录、维保记录、保险记录、沟通记录已入库，但 `归属门店` 为空 |
| `EX-CMP-003` | 业务记录缺归属顾问 | 销售记录、维保记录、保险记录、沟通记录已入库，但 `归属顾问` 为空 |
| `EX-CMP-004` | 金融信息字段缺失 | 存在贷款事实，但 `金融机构`、`贷款周期`、`贷款金额`、`客户费率`、`贷款服务费用`、`车辆上牌服务费`、`车辆上牌城市` 存在缺失 |
| `EX-CMP-005` | 保险信息字段缺失 | 存在保单事实，但 `保单号`、`保险公司`、`起保日期`、`终保日期`、`购买日期`、`保费金额` 存在缺失 |
 
### 5.3.5 关联性异常

| 错误码 | 场景 | 异常出现方式 |
| --- | --- | --- |
| `EX-CMP-009` | 交易记录无法挂接客户 | 销售记录、权益记录、保险记录、活动记录存在，但无法通过 `BPID`、`手机号` 挂接客户主体 |
| `EX-CMP-010` | 交易记录无法挂接车辆 | 销售记录、权益记录、维保记录、保险记录存在，但无法通过 `VIN` 挂接车辆主体 |
