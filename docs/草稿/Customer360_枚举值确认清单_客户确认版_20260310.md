# Customer360 枚举值与字典信息确认清单

- 文档版本：`v1.0`
- 形成日期：`2026-03-10`
- 使用对象：`客户项目负责人 / 业务负责人 / 实施团队`
- 文档目标：`用于确认客户列表、上传模板、校验字典、页面筛选项使用的标准枚举值和标准字典信息`

---

## 1. 确认规则

1. 本文件确认 `标准枚举值` 和 `标准字典信息`。
2. 未在本文件确认的枚举值、字典值，不得进入页面筛选项、上传模板、预校验字典、异常校验字典。
3. 同一个字段只能保留一套标准值，不允许同义词并存。
4. `标签分类`、`分群分类` 需要在本文件确认。
5. `具体标签值`、`具体分群名称` 可由用户在后台自行设定，不纳入本文件确认范围。
6. `关联人角色` 为多值角色标签；同一关联人可同时命中多个角色值，同一主体可存在多个 `送修人`、多个 `使用人`。

---

## 2. 标准枚举值确认清单

| 模块 | 字段 | 标准枚举值 | 字段用途 |
| --- | --- | --- | --- |
| 基础主档 | 客户类型 | 个人车主、公司客户 | 标识主体类型 |
| 基础主档 | 性别 | 男、女、未知 | 标识自然人性别 |
| 基础主档 | 联系偏好 | 电话、短信、微信、不接受联系 | 标识客户接受的触达方式 |
| 关联人员子档 | 关联人角色 | 车主、送修人、联系人、经办人、使用人 | 标识关联人员在主体下承担的业务角色，支持多值 |
| 关联人员子档 | 关系类型 | 本人、家庭成员、企业联系人、企业经办人、企业送修人、企业使用人 | 标识主体与关联人的关系性质 |
| 关联人员子档 | 关联人证件类型 | 身份证、护照、港澳居民来往内地通行证、台湾居民来往大陆通行证、外国人永久居留身份证、其他证件 | 标识关联自然人的证件类型 |

| 关联人员子档 | 地址标签 | 常住地址、工作地址、邮寄地址、企业注册地址、企业联系地址 | 标识联系地址类型 |
| 关联人员子档 | 是否主联系人 | 是、否 | 标识多联系人场景的默认联系对象 |
| 关联人员子档 | 是否默认送修人 | 是、否 | 标识多送修人场景的默认送修对象 |
| 关联人员子档 | 是否默认用车人 | 是、否 | 标识多使用人场景的默认用车对象 |
| 关联人员子档 | 关系状态 | 有效、失效、待确认 | 标识关系是否生效 |

| 标签中心 | 标签分类 | 客户标签、企业标签、关联人运营标签 | 标识标签所属分类；具体标签值由后台配置 |
| 运营分群 | 分群分类 | 客户分群、活动分群 | 标识分群所属分类；具体分群名称由后台配置 |
| 车辆信息 | 车辆状态 | 订车中、订车中-在途、待交付、自用、企业自用、异地用车、维修中、已售、已过户、已转出、已报废 | 标识车辆当前业务状态 |
| 车辆信息 | 签单状态 | 待签、已签、已取消 | 标识销售签单状态 |
| 权益资产 | 项目来源 | 销售、赠送、活动、补发 | 标识权益来源 |
| 权益资产 | 权益类型 | 代金券、保养券、维修券、礼包、套餐、礼遇 | 标识权益资产分类 |
| 权益资产 | 权益状态 | 未使用、已使用、已过期、已作废 | 标识权益当前状态 |
| 维保信息 | 服务类型 | 保养、维修、钣喷 | 标识维保业务类型 |
| 维保信息 | 服务状态 | 待施工、施工中、已完成、已取消 | 标识服务单状态 |
| 保险信息 | 保单类型 | 交强险、商业险、续保单 | 标识保单业务类型 |
| 沟通信息 | 沟通方式 | 电话、微信、短信、现场拜访 | 标识沟通渠道 |
| 沟通信息 | 沟通结果 | 已接通、未接通、已预约、待跟进、已拒绝 | 标识沟通结果 |
